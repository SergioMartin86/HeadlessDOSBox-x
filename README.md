HeadlessDOSBox-x
-----------------

[![Build & Tests](https://github.com/SergioMartin86/HeadlessDOSBox/actions/workflows/make.yml/badge.svg)](https://github.com/SergioMartin86/quickerGambatte/actions/workflows/make.yml)
